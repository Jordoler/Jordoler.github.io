1. Systems of first order ODEs

Assume \(G\) is an open set in \(\mathbb{R} \times \mathbb{R}^n\) (or in \(\mathbb{R} \times \mathbb{C}^n\)) and \(f:G \rightarrow \mathbb{R}^n\) (or \(\mathbb{C}^n\)) is a given vector field.

What does this mean?

To break it down, \(G\) being an open set in \(\mathbb{R} \times \mathbb{R}^n\) (or in \(\mathbb{R} \times \mathbb{C}^n\)) means that there is a space, \(\mathbb{R} \times \mathbb{R}^n\) (or in \(\mathbb{R} \times \mathbb{C}^n\)), that has two components:

  1. \(\mathbb{R}\): A real-valued number that comes from the first \(\mathbb{R}\) in \(\mathbb{R} \times \mathbb{R}^n\) (or in \(\mathbb{R} \times \mathbb{C}^n\)). This is a 1-Dimensional quantity. For example, consider it as the time \(t\) dimension.

  2. \(\mathbb{R}^n\) or \(\mathbb{C}^n\): A real-valued or complex-valued N-Dimensional vector quantity. For example, a position vector \(x\).

An example of this would be \((t, x)\), where \(t\) is from \(\mathbb{R}\) and \(x\) is from \(\mathbb{R}^n\) (or \(\mathbb{C}^n\)).

As \(G\) is an open set of \(\mathbb{R} \times \mathbb{R}^n\) (or in \(\mathbb{R} \times \mathbb{C}^n\)), it contains points in this space but not necessarily the points along the “boundary” of the space.

The vector field \(f\) by definition is: a vector field is an assignment of a vector to each point in a space. Thus, \(f: G \rightarrow \mathbb{R}^n\) (or \(\mathbb{C}^n\)) says that \(f\) assigns a vector in \(\mathbb{R}^n\) (or \(\mathbb{C}^n\)) to every point in \(G\).

If we think of this logical statement in relation to Newton’s equations of motion:

\[ f(t, \vec{r}) = \vec{r} = \frac{d\vec{r}}{dt}t - \frac{1}{2} \frac{d^2\vec{r}}{dt^2}t^2 \]

We see that the function \(f\) here assigns a vector, position, to each point \((t, \vec{r})\) in \(G\).

We are looking for a differentiable vector-function \[x : \langle a,b \rangle \subset \mathbb{R} \rightarrow \mathbb{R}^n \text{ (or } \mathbb{C}^n ), \ \ \ x(t) = \begin{pmatrix} x_1(t) \\ x_2(t) \\ \dots \\ x_n(t) \end{pmatrix} \; \] which satisfies the system of \(n\) ordinary differential equations: \[x'(t) = f(t,x(t)) \iff \begin{cases} \begin{matrix} x_1'(t) = f_1(t,x_1(t),x_2(t), \dots , x_n(t)) \\ x_2'(t) = f_1(t,x_1(t),x_2(t), \dots , x_n(t)) \\ \dots \\ x_n'(t) = f_1(t,x_1(t),x_2(t), \dots , x_n(t)) \end{matrix} \end{cases} \ \ \ \ \ \ \ \ (*)\]

In the sake of brevity we focus on the real case below. In our course we will always assume that \[ f \in C(G;\mathbb{R}^n)\]

What does this mean?

We are looking for a differentiable vector-function that satisfies certain conditions applied on \(x\).

\(x:\langle a,b \rangle \subset \mathbb{R} \rightarrow \mathbb{R}^n\), where: - \(\langle a,b \rangle\) denotes any of the intervals \((a,b)\), \([a,b]\), \([a,b)\), \((a,b]\), \([a,+\infty)\), \((-\infty,+\infty)\), etc. - \(t\) in \(x(t)\) is an element of \(\langle a,b \rangle\), a subset of \(\mathbb{R}\), and its output is a vector in \(\mathbb{R}^n\) (or \(\mathbb{C}^n\)) with \(n\) components.

This vector-function satisfies the system of \(n\) ordinary differential equations:

\[ x'(t) = f(t, x(t)) \iff \begin{cases} x_1'(t) = f_1(t, x_1(t), x_2(t), \dots, x_n(t)) \\ x_2'(t) = f_2(t, x_1(t), x_2(t), \dots, x_n(t)) \\ \vdots \\ x_n'(t) = f_n(t, x_1(t), x_2(t), \dots, x_n(t)) \end{cases} \ \ \ \ \ \ \ (*) \]

This defines another condition applied to \(x\): the derivative of \(x\) is a function \(f\) that maps the inputs \(t\) and \(x(t)\) to another vector:

\[ f(t, x(t)) = \begin{pmatrix} f_1(t, x_1(t), x_2(t), \dots, x_n(t)) \\ f_2(t, x_1(t), x_2(t), \dots, x_n(t)) \\ \vdots \\ f_n(t, x_1(t), x_2(t), \dots, x_n(t)) \end{pmatrix} = x'(t) \]

Note here that \(f(t, x(t)) = x'(t)\) makes \(x(t)\) a differentiable function. Furthermore, the condition that this differential function must be an Ordinary Differential Equation is placed on \(f(t, x(t)) = x'(t)\).

Finally, \(f \in C(G; \mathbb{R}^n)\) states that the function \(f\) is continuous and nice.

2. Definition of a solution

\(D\tiny{EFINITION}\). We say that \(x:\langle a,b \rangle \rightarrow \mathbb{R}^n\) is a solution to the system if (\(*\)) iff

  1. \(x \in C^1(\langle a,b \rangle;\mathbb{R}^n\)

  2. \((t,x(t)) \in G, \ \ \ \forall t \in \langle a,b \rangle\)

  3. \(x'(t) = f(t,x(t)), \ \ \ \forall t \in \langle a,b \rangle\)

What does this mean?

This point simply states that a differentiable vector-function \(x\) is a solution to the system of \(n\) ordinary differential equations (\(*\)) if and only if:

  1. \(x\), with \(n\) components, is continuously differentiable across the interval \(\langle a, b \rangle\).

  2. For all \(t\) that is an element in the interval \(\langle a, b \rangle\), \(t\) and \(x(t)\) are an element of \(G\).

    Remember: \(G\) is an open set in \(\mathbb{R} \times \mathbb{R}^n\) (or in \(\mathbb{R} \times \mathbb{C}^n\)).

  3. For all \(t\) that is an element in the interval \(\langle a, b \rangle\), the derivative of \(x(t)\), \(x'(t)\), is equal to the function \(f(t, x(t))\).

    This condition seems trivial since \(f(t, x(t))\) is defined by \(f(t, x(t)) = x'(t)\). However, it also places the restriction on \(x(t)\) that it must be a function.
