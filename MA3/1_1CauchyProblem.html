1. Systems of first order ODEs

Assume \(G\) is an open set in \(\mathbb{R} \times \mathbb{R}^n\) (or in \(\mathbb{R} \times \mathbb{C}^n\)) and \(f:G \rightarrow \mathbb{R}^n\) (or \(\mathbb{C}^n\)) is a given vector field.

What does this mean?

Remark 1.

\(G\) being an open set in \(\mathbb{R} \times \mathbb{R}^n\) (or in \(\mathbb{R} \times \mathbb{C}^n\)) means that there is a space, \(\mathbb{R} \times \mathbb{R}^n\) (or in \(\mathbb{R} \times \mathbb{C}^n\)), that has two components:

  1. \(\mathbb{R}\): A real-valued number that comes from the first \(\mathbb{R}\) in \(\mathbb{R} \times \mathbb{R}^n\) (or in \(\mathbb{R} \times \mathbb{C}^n\)). This is a 1-Dimensional quantity. For example, consider it as the time \(t\) dimension.

  2. \(\mathbb{R}^n\) or \(\mathbb{C}^n\): A real-valued or complex-valued N-Dimensional vector quantity. For example, a position vector \(x\).

An example of this would be \((t, x)\), where \(t\) is from \(\mathbb{R}\) and \(x\) is from \(\mathbb{R}^n\) (or \(\mathbb{C}^n\)).

As \(G\) is an open set of \(\mathbb{R} \times \mathbb{R}^n\) (or in \(\mathbb{R} \times \mathbb{C}^n\)), it contains points in this space but not necessarily the points along the “boundary” of the space.

The vector field by definition: a vector field is an assignment of a vector to each point in a space. Thus, \(f: G \rightarrow \mathbb{R}^n\) (or \(\mathbb{C}^n\)) says that \(f\) assigns a vector in \(\mathbb{R}^n\) (or \(\mathbb{C}^n\)) to every point in \(G\).

If we think of this logical statement in relation to Newton’s equations of motion:

\[ f(t, \vec{r}) = \vec{r} = \frac{d\vec{r}}{dt}t - \frac{1}{2} \frac{d^2\vec{r}}{dt^2}t^2 \]

We see that the function \(f\) here assigns a vector, position, to each point \((t, \vec{r})\) in \(G\).

We are looking for a differentiable vector-function \[ x : \langle a,b \rangle \subset \mathbb{R} \rightarrow \mathbb{R}^n \text{ (or } \mathbb{C}^n ), \ \ \ x(t) = \begin{pmatrix} x_1(t) \\ x_2(t) \\ \dots \\ x_n(t) \end{pmatrix} \; \tag{1}\] which satisfies the system of \(n\) ordinary differential equations: \[ x'(t) = f(t,x(t)) \iff \begin{cases} \begin{matrix} x_1'(t) = f_1(t,x_1(t),x_2(t), \dots , x_n(t)) \\ x_2'(t) = f_1(t,x_1(t),x_2(t), \dots , x_n(t)) \\ \dots \\ x_n'(t) = f_1(t,x_1(t),x_2(t), \dots , x_n(t)) \end{matrix} \end{cases} \tag{2}\]

In the sake of brevity we focus on the real case below. In our course we will always assume that \[ f \in C(G;\mathbb{R}^n) \]

What does this mean?

We are looking for a differentiable vector-function that satisfies certain conditions applied on \(x\).

\(x:\langle a,b \rangle \subset \mathbb{R} \rightarrow \mathbb{R}^n\), where: - \(\langle a,b \rangle\) denotes any of the intervals \((a,b)\), \([a,b]\), \([a,b)\), \((a,b]\), \([a,+\infty)\), \((-\infty,+\infty)\), etc. - \(t\) in \(x(t)\) is an element of \(\langle a,b \rangle\), a subset of \(\mathbb{R}\), and its output is a vector in \(\mathbb{R}^n\) (or \(\mathbb{C}^n\)) with \(n\) components.

This vector-function satisfies the system of \(n\) ordinary differential equations:

\[ x'(t) = f(t, x(t)) \iff \begin{cases} x_1'(t) = f_1(t, x_1(t), x_2(t), \dots, x_n(t)) \\ x_2'(t) = f_2(t, x_1(t), x_2(t), \dots, x_n(t)) \\ \vdots \\ x_n'(t) = f_n(t, x_1(t), x_2(t), \dots, x_n(t)) \end{cases} \ \ \ \ \ \ \ (2) \]

This defines another condition applied to \(x\): the derivative of \(x\) is a function \(f\) that maps the inputs \(t\) and \(x(t)\) to another vector:

\[ f(t, x(t)) = \begin{pmatrix} f_1(t, x_1(t), x_2(t), \dots, x_n(t)) \\ f_2(t, x_1(t), x_2(t), \dots, x_n(t)) \\ \vdots \\ f_n(t, x_1(t), x_2(t), \dots, x_n(t)) \end{pmatrix} = x'(t) \]

Note here that \(f(t, x(t)) = x'(t)\) makes \(x(t)\) a differentiable function. Furthermore, the condition that this differential function must be an Ordinary Differential Equation is placed on \(f(t, x(t)) = x'(t)\).

Finally, \(f \in C(G; \mathbb{R}^n)\) states that the function \(f\) is continuous and nice.

2. Definition of a solution

\(D\tiny{EFINITION}\). We say that \(x:\langle a,b \rangle \rightarrow \mathbb{R}^n\) is a solution to the system Equation 2 iff

  1. \(x \in C^1(\langle a,b \rangle;\mathbb{R}^n)\)

  2. \((t,x(t)) \in G, \ \ \ \forall t \in \langle a,b \rangle\)

  3. \(x'(t) = f(t,x(t)), \ \ \ \forall t \in \langle a,b \rangle\)

What does this mean?

This point simply states that a differentiable vector-function \(x\) is a solution to the system of \(n\) ordinary differential equation, Equation 2, if and only if:

  1. \(x\), with \(n\) components, is continuously differentiable across the interval \(\langle a, b \rangle\).

  2. For all \(t\) that is an element in the interval \(\langle a, b \rangle\), \(t\) and \(x(t)\) are an element of \(G\).

    Remember: \(G\) is an open set in \(\mathbb{R} \times \mathbb{R}^n\) (or in \(\mathbb{R} \times \mathbb{C}^n\)).

  3. For all \(t\) that is an element in the interval \(\langle a, b \rangle\), the derivative of \(x(t)\), \(x'(t)\), is equal to the function \(f(t, x(t))\).

    This condition seems trivial since \(f(t, x(t))\) is defined by \(f(t, x(t)) = x'(t)\). However, it also places the restriction on \(x(t)\) that it must be a function.

3. Cauchy Problem

We will also consider the Cauchy Problem for the system (\(2\)): for given \(x_0 \in \mathbb{R}^n\) and \(t_0 \in \langle a,b \rangle\) such that \((t_0,x_0) \in G\) for a solution \(x \in C^1(\langle a,b \rangle;\mathbb{R}^n)\) of the system, Equation 2 , such that: \[ \begin{cases} \begin{matrix} x'(t) = f(t,x(t)), \ \ \ \ t\in \langle a,b\rangle \\ x(t_0) = x_0 \end{matrix} \end{cases} \tag{3}\]

What does this mean?

The most difficult part of this point is the term Cauchy Problem perhaps its easier to think of it as it’s other, more popular name The Initial Value Problem. where there is some initial value \(x(t_0) = x_0\) with \(t_0\) an element of an arbitrary interval \(\langle a,b \rangle\) such that the “point” \((t_0,x_0)\) is an element of \(G\) Remark 1.

for a solution \(x \in C^1(\langle a,b \rangle;\mathbb{R}^n)\) of the system (2)

for a solution \(x\) which is a function that is continuously differentiable along the arbitary domain \(\langle a,b \rangle\) such that it is in \(\mathbb{R}^n\). recall that \(x\) here references the function, Equation 1 , previously defined and . more conditions on \(x\) are then applied \(x'(t) = f(t,x(t)), \ \ \ t\in \langle a,b \rangle\) and \(x(t_0)=x_0\). if we compare these conditions with Equation 2 we see that the solution the Cauchy Problem Initial Value Problem has the derivative \(x'(t)\) that is equal to Equation 2.

4. Reduction of the Cauchy problem to the integral equation

\(T\tiny{HEOREM}\) Assume \(x_0 \in \mathbb{R}^n\) and \(t_0 \in \langle a,b \rangle\) satisfy \((t_0,x_0) \in G\). A function \(x : \langle a,b \rangle \rightarrow \mathbb{R}^n\) is a solution to the Cauchy problem, Equation 3, if and only if:

  1. \(x \in C(\langle a,b \rangle;\mathbb{R}^n)\)
  2. \((t,x(t)) \in G, \ \ \ \forall t \in \langle a,b \rangle\)
  3. The function \(x(t)\) satisfies the integral equation

\[x(t) = x_0 + \int_{t_0}^tf(\tau,x(\tau)) \ d\tau , \ \ \ \forall t \in \langle a,b \rangle\]

What does this mean?

Points 1 and 2 have been covered before, a quick recap is: \(x\) is continuous along the arbitary interval \(\langle a,b \rangle\) such that it is in \(\mathbb{R}^n\). and the “point”\((t,x(t))\) is a subset of \(G\), Remark 1, for \(t\) along the arbitary interval \(\langle a,b \rangle\).

The third point applies the condition on \(x(t)\) such it is equal to the integral of \(f(t,x(t))\), Equation 2, from \(t_0\) to \(t\), and has been parametrized by \(\tau\) where \(t\) is in the arbitary interval \(\langle a,b \rangle\).

5. Peano’s interval

Assume \((t_0,x_0) \in G\) and \(r >0\) and \(R>0\) satisfy: \[ \bar{P}_{r,R}(t_0,x_0) := [t_0 - r,t_0+r] \times \bar{B}_R(x_0) \subset G \] As \(\bar{P}_{r,R}(t_0,x_0)\) is compact and \(f \in C(\bar{P}_{r,R}(t_0,x_0))\) we can define \[ M:=\sup\limits_{(t,x)\in \bar{P}_{r,R}(t_0,x_0)} |f(t,x)|. \] Introduce a positive number \[ h := \min \left\{ {r,\frac{R}{M}}\right\} \]

Then the interval \([t_0-h,t_0+h]\) is called the Peano Interval for \(f\) and \(\bar{P}_{r,R}(t_0,x_0)\).

\(T\tiny{HEOREM}.\) If \(x(t)\) is a solution to the Cauchy problem (\(**\)) and \([t_0 - h,t_0+h]\) we have \(x(t) \in B_R(x_0)\).

6. Euler method

Assume \(P_{r,R}(t_0,x_0) \subset G\) and let \([t_0-h, t_0+h]\) be the Peano interval for \(f\) and \(\bar{P}_{r,R}(t_0,x_0)\). For any \(m \in \mathbb{N}\) we split the interval \([t_0,h]\) into \(m\) parts by points \[ t_0 < t_1 < t_2 < \ ... \ < t_m = t_0 + h, \ \ \ \ \ t_k=t_0+\frac{k}{m}h, \ \ k = 0,1,...,m \]

then we define points \({\{x_k\} }_{k=1}^{m} \subset \mathbb{R}^n\) recursively as solutions of the following equation:

\[ \frac{x_k - x_{k-1}}{t_k - t_{k-1}} = f(t_{k-1},x_{}k-1), \ \ \ \ k=1,2,...,m, \ \ \ \ t_k - t_{k-1}=\frac{h}{m}. \] In other words, we define points, \(x_1,x_2,...,x_m\) in the following recursive way: \[ x_1 = x_0 + f(t_0,x_0)\frac{h}{m}, \ \ \ x_2 = x_1 + f(t_1,x_1)\frac{h}{m}, \ \ \ ... \ \ \ x_m = x_{m-1} + f(t_{m-1},x_{m})\frac{h}{m}. \] Then for any \(t \in [t_0,h]\) we define the forward Euler approximations \(\varphi_m(t)\) for the Cauchy problem (\(**\)) as a piece-wise linear continuous function: \[ \varphi_m(t) := x_{k-1} + \frac{x_k - x_{k-1}}{t_k - t_{k-1}}(t-t_{k-1}), \ \ \ \ \forall t \in [t_{k-1},t_k], \ \ \ \ k=1,2,...,m \] similar we can define the backward Euler approximations \(\varphi_m(t)\) on the interval \([t_0 - h, t_0]\).

7. Compactness of Euler approximations

\(T\tiny{HEOREM}.\) Assume \(\bar{P}_{r,R}(t_0,x_0) \subset G\) and let \([t_0-h,t_0+h]\) be the Peano interval for \(f\) and \(\bar{P}_{r,R}(t_0,x_0)\). Assume \(\varepsilon_m \rightarrow +0\).

Then for any \(m \in \mathbb{N}\) there exists \(N=N(m) \in \mathbb{N}\) such that the EUler approximations \(\varphi_m \in C([t_0 - h,t_0+h];\mathbb{R}^n)\) for the Cauchy problem, Equation 3, (corresponding to the mesh of partition \(\frac{h}{}\)) possess the following properties:

  1. \(\varphi_m(t) \in \bar{B}_R(x_0) , \ \ \ \ \forall t \subset [t_0 - h, t_0 + h]\)
  2. \(|\varphi_m'(t) - f(t,\varphi_m(t))| \le \varepsilon_m , \ \ \ \ \forall t \in [t_0 - h, t_0 + h]\)
  3. functions \(\{ \varphi_m \}_{m=1}^{\infty}\) are uniformly bounded on \([t_0 - h, t_0 + h]\)
  4. functions \(\{ \varphi_m \}_{m=1}^{\infty}\) are uniformly equicontinuous on \([t_0 - h, t_0 + h]\)
